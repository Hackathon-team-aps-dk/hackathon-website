How It All Started

About Project:



Project Ridgewood is a Portfolio Manager that was made under 24 hours for a hackathon it aims to help and guide people choose where to invest their money. The website only requires the salary , assets and liabilities and it divides the best places to where and how much you should invest your capital with certain guides. It is best for people new to investing as it avails alot of channels where people can spend their money.



About website:

The Website is written in javascript , html and css and uses php as a worker for the website hosting. The website was made only after reading reknowned investing articles on the internet and based on them tells where and how much you should invest it uses jquery as a library for animations the website has lots of features including sitemaps , crawlers , indexed pages etc.
